# Duey

Duey는 Android용 할 일/일정 관리 앱입니다.  
기존에 많은 Todo 앱들은 특정 날짜를 기준으로 Todo를 관리하지만, Duey는 기간 단위로 Todo를 관리하여 특정 기간을 정해두고 원하는 날에 완료 처리를 할 수 있도록 설계되었습니다.

## 기술 스택

- Android Gradle Plugin `9.1.1`
- Kotlin `2.3.21`
- Java `17`
- Compile SDK `37`, Min SDK `26`, Target SDK `35`
- Jetpack Compose BOM `2026.04.01`
- Material 3
- Room `2.8.4`
- KSP `2.3.6`
- Glance AppWidget `1.1.1`
- Spotless `8.4.0`, ktlint `1.5.0`
